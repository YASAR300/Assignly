# Generated Samples — Assignly AI Studio Demo Output

This folder contains **8 AI-generated product images** produced by the Assignly AI Studio pipeline from a single pearl jewelry reference photograph.

All 8 images were generated by running:

```bash
cd backend
python generate_samples_script.py
```

## ⚙️ How the Pipeline Works

```
original_reference.png
        │
        ▼
  rembg (onnxruntime)   ──→  extracted_product.png  (transparent PNG)
        │
        ├── × 8 background types
        │         │
        │    [Sandbox Mode]  ──  Curated Unsplash photos / solid color
        │    [AI Mode]       ──  Stability AI / Replicate / HuggingFace
        │         │
        ▼         ▼
   PIL Compositor
   - Smart scaling (55–70% of 800px canvas)
   - Centred placement
   - Realistic drop-shadow (GaussianBlur alpha mask)
        │
        ▼
   800 × 800 JPEG  (quality=95)
```

> **Product consistency guarantee:** The product foreground is extracted exactly once and re-composited onto each unique background. The jewel, its reflections, and its details are pixel-for-pixel identical across all 8 outputs.

---

## 🔑 AI Generation Modes

The pipeline automatically selects the best available generation engine:

1. **Stability AI** — if `STABILITY_API_KEY` is set in `.env`
2. **Replicate** — if `REPLICATE_API_TOKEN` is set
3. **Hugging Face** — if `HF_API_TOKEN` is set
4. **Sandbox Mode** *(default)* — curated royalty-free Unsplash images; no API key required

The samples in this folder were generated using **Sandbox Mode**.
